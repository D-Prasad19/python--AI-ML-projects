# Handedness & Mortality: A Bayesian Statistical Analysis 🧬🔢

Do left-handed people really die younger? This project uses **Bayesian Statistics** and **Python** to investigate the "death age gap" and refute common misconceptions using historical data.

## 🧐 The Problem
Older studies suggested left-handers die earlier. This project proves that this is a statistical illusion caused by the changing social acceptance of left-handedness over the 20th century.

## 🛠️ Tech Stack & Concepts
- **Language:** Python
- **Libraries:** Pandas, NumPy, Matplotlib
- **Mathematical Focus:** Bayesian Statistics, Probability Distributions, Extrapolation.

## 📊 Methodology
1. **Data Integration:** Combined National Geographic survey data (1986) with death distribution data.
2. **Statistical Modeling:** Calculated the probability of being a certain age given handedness: $P(A | LH)$.
3. **Refutation:** Successfully reproduced the "age gap" using data trends, showing it is a result of birth-year rates rather than biological longevity.
